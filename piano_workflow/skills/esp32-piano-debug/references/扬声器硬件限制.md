# 扬声器/蜂鸣器硬件限制

## 板载驱动电路（手册 4.1 节）

```
VCC(3.3V) ── 负载(板载蜂鸣器 B1 / 外接喇叭) ── Q3 集电极
GPIO25 ── J11(跳线) ── R13(基极限流) ── Q3 基极    (Q3 = MMSS8050 NPN)
GND ── Q3 发射极
```

- 共射极开关：GPIO25 **高电平 → Q3 导通 → 负载通电**
- J11 默认插跳线帽；断开即隔离 GPIO25 与该电路
- 板载是**无源蜂鸣器**：靠 PWM 频率定音高，改占空比≠改音量（压电特性）

## 外接 8Ω 喇叭的硬约束（来自 KEY.py 实测注释）

1. **绝不可把 8Ω 喇叭直接接 GPIO25↔GND**：
   峰值电流 ≈ 3.3V / 8Ω ≈ **0.41A**，ESP32 单脚极限约 40mA，直连会
   损坏 IO 或整片。占空比限流（`POWER_NUM/POWER_DEN`）只能降低**平均**
   导通时间，**降不了高电平瞬间的峰值电流**。
2. 正确接法（任选其一）：
   - 走板载 Q3（喇叭替换/并接 B1 位置，电流由 Q3 承担）
   - 串联限流电阻（响度换安全）
   - 专用音频功放（PAM8403 等，效果最好）
3. **建议加隔直电容**：PWM-DAC 有直流偏置（CENTER 占空比），直流分量
   会让音圈持续偏移发热、损失冲程。
4. **低频冲程保护**：降低调制深度（如 `MOD_TICKS` 从满摆幅降到 32）
   可让小喇叭低音更平顺、不拍底。

## 功率/占空比参数含义（KEY.py 方案）

- `TICKS=125, CENTER=62`：PWM 周期与零振幅中心
- `POWER_NUM/POWER_DEN = 1/5`：实际高电平时间 = 合成 duty × 1/5，
  零振幅附近 Q3 平均导通 ≈ (62/125)/5 ≈ 9.9%
- 音量调节的正确手段是调制幅度（AMP/MOD_TICKS）和功率分频比，
  **不是**提高电源电压或去掉限流

## 防爆音/防哒声（软件侧，但根源是硬件）

- **爆音**：引脚从长低电平瞬间跳到 CENTER 占空比 = 阶跃直流 → 膜片
  猛弹。首尾必须做 ~10ms 的占空比线性斜坡（0↔CENTER）。
- **哒哒音**：RMT 分块播放块间恒有 35µs 低电平空隙，等效丢失
  ≈233tick×偏置的电荷，膜片感知为周期性凹坑。深度与音量无关，音尾
  衰减后反而更明显。对策：
  - 周期波形（纯音）：`loop(True)` 硬件循环，零接缝
  - 非周期波形（带包络）：分块流水 + **块尾采样高电平 +233tick 电荷
    补偿**，凹坑与补偿相邻 140µs 内经膜片低通相抵
  - 播放期间零 flash 读取、零大对象分配（双缓冲就地填充），防 GC
    停顿拉大空隙
