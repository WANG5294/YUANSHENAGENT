### 功能实现

| 按键 | 功能 |
|------|------|
| **KEY1** (GPIO35) | 流水灯循环：G05→G12→G14→G18→G19→G21→G22→G23→G05...，每个亮1秒 |
| **BOOT** (GPIO0) | 停止当前模式，返回主循环 |
| **KEY2** (GPIO34) | 计时亮灯：按下后秒数 % 8 = 亮灯数量（0s→0灯，1s→1灯...7s→7灯，8s→0灯） |
