# ESP32-D0WD 板引脚分区速查

来源：`修正ESP32_D0WD_硬件开发手册.md` §三/§五/§8.10（网表实测）。

## GPIO 总表

| GPIO | 板上用途 | 可用性 | 关键约束 |
|:---:|---|:---:|---|
| 0 | BOOT 键(K1)、J5_15/16 | ⚠️ | 低电平进下载模式；建议 PULL_UP |
| 1/3 | UART0 ↔ CP2102N | ⚠️ | MicroPython REPL 占用，勿他用 |
| 2 | 排针 J7 | ✅ | 启动时需浮空或低 |
| 4 | 排针 J8 | ✅ | — |
| 5 | 排针 J4_1/2 | ✅ | Strapping（一般无碍） |
| 6~11 | SPI Flash (ZB25VQ32) | ❌ | 绝对不可用 |
| 12 | 排针 J4_3/4 | ⚠️ | MTDI：启动时必须低，外接上拉会无法启动 |
| 13 | 排针 J9 | ✅ | — |
| 14 | 排针 J4_5/6 | ✅ | — |
| 15 | 排针 J10 | ⚠️ | MTDO：启动时必须高 |
| 16/17 | MPU6050 SDA/SCL | ✅ | 板载 10k 上拉（R7/R8） |
| 18/19/21/22/23 | 排针 J4 | ✅ | 通用，VSPI/I2C0 默认脚在此 |
| 25 | 蜂鸣器(经J11)、J5_5/6 | ✅ | DAC1；详见 esp32-buzzer 技能 |
| 26/27 | MX620B 电机 IN1/IN2 | ✅ | 26 是 DAC2 |
| 32 | LED2 绿、J5_13/14 | ✅ | 低电平点亮；详见 esp32-led |
| 33 | LED3 红、J5_11/12 | ✅ | 低电平点亮 |
| 34 | KEY2、J5_9/10 | ✅ | **仅输入**，板载 1k 上拉(R9)；详见 esp32-keys |
| 35 | KEY1、J5_7/8 | ✅ | **仅输入**，板载 1k 上拉(R12) |

## 特殊功能分组

- **仅输入（无输出、无内部上下拉）**：GPIO34、35
- **Strapping**：GPIO0、2、5、12、15（启动电平敏感）
- **ADC1**（Wi-Fi 下仍可用）：32/33/34/35；**ADC2**（Wi-Fi 下不可用）：4/12/13/14/15/27
- **DAC**：25(DAC1)、26(DAC2)；**触摸**：2/4/12~15/27/32/33
- **PWM(LEDC)/RMT/中断**：全部输出可用 GPIO 均支持

## 冲突矩阵

| 若使用… | 则不可… | 原因 |
|---|---|---|
| Wi-Fi | ADC2 (4/12~15/27) | 与射频共用 |
| REPL | GPIO1/3 做普通 IO | UART0 占用 |
| GPIO25 他用 | 保留 J11 跳线 | 蜂鸣器会跟着响，先拔 J11 |
| GPIO12 外接上拉 | 正常启动 | MTDI 必须低 |
| GPIO15 外接下拉 | 正常启动日志 | MTDO 必须高 |

## 排针速查

- J1=VCC(3.3V)×8、J2=+5V×8、J3=GND×8
- J4（左 16P）：5/12/14/18/19/21/22/23（每 GPIO 占相邻两针）
- J5（右 16P）：17/16/25/35/34/33/32/0
- J7=GPIO2+VCC+GND、J8=GPIO4+VCC+GND、J9=GPIO13+5V+GND、J10=GPIO15+5V+GND
- J6=电机（GND/A1/A2/BAT）、J11=蜂鸣器跳线（R13↔GPIO25）

## 电气极限

单脚吸入 28mA / 输出 40mA，**长期建议 ≤12mA**；RTC 与 CPU 两个电源域各 40mA 总限。8Ω 喇叭等大电流负载严禁直接接 GPIO。
