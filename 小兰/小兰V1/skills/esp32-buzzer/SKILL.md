# 蜂鸣器（MLT-5020 压电式，GPIO25）

## 驱动电路

```
VCC ── B1(蜂鸣器) ── Q3集电极(MMSS8050 NPN)
GPIO25 ── J11(跳线) ── R13(基极限流) ── Q3基极
GND ── Q3发射极
```

- NPN 共射极开关：GPIO25 **高电平 → Q3 导通 → 发声**
- **J11 跳线**默认插着；拔掉即隔离 GPIO25 与蜂鸣器（GPIO25 他用/做 DAC 前必拔）
- 无源压电蜂鸣器：**PWM 频率定音高，占空比改不了音量**

## MicroPython 用法

```python
from machine import Pin, PWM
buzzer = PWM(Pin(25), freq=440, duty=0)   # 初始静音
buzzer.duty(512)    # 50% 占空比发声（10-bit 0~1023）
buzzer.duty(0)      # 静音
buzzer.deinit()     # 用完释放，且写低电平防 Q3 持续导通
```

## 音阶频率表（A4=440Hz 十二平均律）

| 音名 | C4 | D4 | E4 | F4 | G4 | A4 | B4 | C5 |
|---|---|---|---|---|---|---|---|---|
| Hz | 262 | 294 | 330 | 349 | 392 | 440 | 494 | 523 |

升八度 ×2、降八度 ÷2；半音 ×2^(1/12)。

## 安全与音质要点

1. **外接 8Ω 喇叭严禁直连 GPIO25↔GND**：峰值电流 ≈0.41A，远超单脚 40mA 极限。
   正确接法：走板载 Q3（替换/并接 B1）、串限流电阻、或 PAM8403 等功放。
2. 外接喇叭建议加**隔直电容**，PWM 直流偏置会让音圈发热失冲程。
3. 防爆音：从静音到发声/结束时避免电平阶跃，可做 ~10ms 占空比斜坡。
4. GPIO25 同时是 DAC1；本板高级音频方案（RMT PWM-DAC）见项目根目录
   `RMT音频哒哒音问题说明.md`。
